# Tableau de bord

Après connexion, vous arrivez sur votre tableau de bord — vue d'ensemble de votre activité.

## Éléments affichés

| Élément | Description |
|---|---|
| **Postes actifs** | Nombre de postes avec un profil Big Five défini |
| **Équipes** | Nombre d'équipes créées |
| **Questionnaires en attente** | Liens envoyés mais pas encore complétés |
| **Rapports récents** | Derniers rapports de fit générés |

## Navigation principale

La barre de navigation donne accès à toutes les sections :

- **Postes** — gérer vos fiches de poste et leurs profils cibles
- **Équipes** — gérer vos équipes et leurs membres
- **Questionnaires** — suivre les envois et la progression
- **Rapports** — consulter et exporter les rapports de fit
