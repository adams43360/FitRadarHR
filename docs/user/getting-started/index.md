# Prise en main

Bienvenue dans FitRadarHR. Cette section vous guide de la création de compte jusqu'à votre premier rapport de fit.

## Étapes de démarrage

1. [Créer un compte](signup.md) — choisissez entre un compte organisation (B2B) ou personnel
2. [Découvrir le tableau de bord](dashboard.md) — aperçu des fonctionnalités disponibles
3. Créer votre premier [poste](../positions/create.md) ou [équipe](../teams/create.md)
4. [Envoyer un questionnaire](../survey/send.md) à un candidat ou collaborateur
5. Consulter le [rapport de fit](../reports/index.md)

## Prérequis

- Un navigateur web récent (Chrome, Firefox, Safari, Edge)
- Une adresse email valide pour la création de compte
- Une adresse email de la personne à évaluer (pour l'envoi du questionnaire)
