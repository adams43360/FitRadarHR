# Créer une équipe

## Accès

Depuis le tableau de bord ou le menu **Équipes**, cliquez sur **Nouvelle équipe**.

<!-- screenshot: docs/user/assets/team-create.png -->
!!! note "Capture d'écran à venir"

## Champs du formulaire

| Champ | Obligatoire | Description |
|---|---|---|
| Nom de l'équipe | ✅ | Ex. : *Équipe produit*, *Sales EMEA* |
| Description | ❌ | Contexte, missions de l'équipe |
| Manager responsable | ✅ | Utilisateur de type Manager ou Solo dans votre organisation |

## Après la création

L'équipe est créée sans membres. L'étape suivante est d'[ajouter des membres](members.md) pour que le Fit Équipe puisse être calculé.
