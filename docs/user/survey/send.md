# Envoyer un lien questionnaire

## Accès

Depuis la fiche d'une personne (candidat ou collaborateur), cliquez sur **Envoyer le questionnaire** — l'email est pré-rempli automatiquement.

Vous pouvez aussi accéder directement au formulaire via **Questionnaires → Envoyer un lien**.

## Options d'envoi

| Champ | Obligatoire | Description |
|---|---|---|
| Email | ✅ | Email de la personne |
| Prénom / Nom | Si nouvelle personne | Remplis si la personne n'existe pas encore — elle sera créée automatiquement |
| Poste concerné | Non | Rattache le questionnaire à un poste pour contextualiser le fit dans les rapports |
| Version | — | 50 ou 100 items (50 recommandé — environ 10 min) |
| Langue | — | FR ou EN — la personne peut changer à l'ouverture du lien |

!!! tip "Création à la volée"
    Si vous saisissez un email inconnu, FitRadarHR crée automatiquement la fiche personne avec le prénom et nom que vous avez renseignés. Pas besoin de passer d'abord par la section Personnes.

!!! tip "Rattachement à un poste"
    En sélectionnant un poste, le rapport de profil du candidat affichera le Fit Poste correspondant en priorité. Cela n'empêche pas le calcul de fit sur les autres postes.

## Ce que reçoit la personne

L'email envoyé contient :

- Un lien sécurisé et unique (non réutilisable)
- La durée estimée du questionnaire
- Une mention RGPD (les données sont utilisées uniquement pour cette évaluation)

!!! warning "Lien à usage unique"
    Chaque lien est associé à une seule personne et ne peut pas être partagé. Si la personne a besoin d'un nouveau lien (lien expiré, email perdu), utilisez le bouton **Renvoyer** dans le dashboard questionnaires.
