# Envoyer un lien questionnaire

## Accès

Depuis la fiche d'une personne (candidat ou collaborateur), cliquez sur **Envoyer le questionnaire**.

<!-- screenshot: docs/user/assets/survey-send.png -->
!!! note "Capture d'écran à venir"

## Options d'envoi

| Option | Valeur par défaut | Description |
|---|---|---|
| Version | Paramètre de l'org (50 ou 100 items) | Modifiable à chaque envoi |
| Langue | Langue par défaut de l'org | La personne peut changer de langue à l'ouverture du lien |
| Expiration | 30 jours | Le lien devient inactif passé ce délai |

## Ce que reçoit la personne

L'email envoyé contient :
- Un lien sécurisé et unique (non réutilisable)
- La durée estimée du questionnaire
- Une mention RGPD (les données sont utilisées uniquement pour cette évaluation)

!!! warning "Lien à usage unique"
    Chaque lien est associé à une seule personne et ne peut pas être partagé. Si la personne a besoin d'un nouveau lien (lien expiré, email perdu), vous pouvez en générer un nouveau depuis son profil.

## Ajouter une personne

Si la personne n'existe pas encore dans FitRadarHR, vous pouvez l'ajouter depuis la section **Personnes** :
- Prénom, nom, email
- Type : Candidat ou Collaborateur
