# Suivre la progression

## Statuts d'un questionnaire

| Statut | Signification |
|---|---|
| ⏳ **En attente** | Lien envoyé, questionnaire non commencé |
| 🔄 **En cours** | La personne a commencé mais n'a pas terminé |
| ✅ **Complété** | Questionnaire terminé, profil Big Five calculé |
| ⛔ **Expiré** | Le lien a dépassé sa date d'expiration |

## Relancer une personne

Si le statut est **En attente** depuis plusieurs jours, vous pouvez renvoyer un email de rappel depuis la fiche de la personne. Le même lien est renvoyé (la date d'expiration reste inchangée).

## Reprendre un questionnaire en cours

La personne peut fermer son navigateur à tout moment. Sa progression est sauvegardée automatiquement. En rouvrant le lien, elle reprend là où elle s'est arrêtée.

!!! info "Confidentialité des réponses"
    Vous ne voyez pas les réponses individuelles aux questions — uniquement les 5 scores dimensionnels du profil final. Les réponses brutes sont supprimées après le calcul du profil.

## Re-passation & suivi longitudinal

Un profil Big Five peut évoluer (mobilité interne, prise de poste, plusieurs années entre deux évaluations). Le bouton **↻ Repasser le questionnaire**, disponible sur le rapport de profil d'une personne, envoie un nouveau lien de passation.

Quand la personne termine cette nouvelle passation, son profil précédent est automatiquement archivé — il n'est jamais perdu. Le rapport de profil affiche alors une section **Évolution du profil** : une courbe et un tableau montrant les scores de chaque dimension au fil des passations successives.

!!! note
    Seul le profil courant (le plus récent) est utilisé pour les calculs de fit avec les postes et les équipes. L'historique sert uniquement au suivi dans le temps.
